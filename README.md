# Cat's Todo Ledger 🐾

A cute cat-themed todo app with pomodoro timer, shop, and stats.

## Features

- 📋 Task management with priority levels and due dates
- 📊 Stats and weekly progress tracking
- 🍅 Pomodoro focus timer
- 🏪 Shop with cat collection and wishlist
- 🪙 Coin reward system
